SVG is an image format for vector graphics.
It literally means Scalable Vector Graphics.
Basically, what you work with in Adobe Illustrator.

Why use SVG at all?

  • Small file sizes that compress well
  • Scales to any size without losing clarity (except very tiny)
  • Looks great on retina displays
  • Design control like interactivity and filters

The blend-mode CSS data type describes how colors should appear when elements overlap. It is used in the background-blend-mode and mix-blend-mode properties.

On this slide we can choose blending mode from the list options.
For example multiply blend mode will show final colors as the result of multiplying the top and bottom colors.
And difference blend mode will show final colors as the result of subtracting the darker of the two colors from the lighter one.

SVG Coordinate System

The viewport
In the SVG coordinate system the point x=0, y=0 is the upper left corner. The y-axis is thus reversed compared to a normal graph coordinate system. As y increases in SVG, the points, shapes etc. move down, not up.

The viewport is the viewing area where the SVG will be visible. You can think of the viewport as a window through which you can see a particular scene.
