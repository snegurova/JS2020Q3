Инструкция по работе с калькулятором Читать далее

Ввод отрицательных значений
осуществляется с помощью кнопки "+/-". Т.е. сначала вводим число, потом нажимаем "+/-" для смены знака, аналогично работает если мы хотим поменять знак для второго операнда. Например, чтобы посчитать "2 + -2" кликаем последовательно "2" "+" "2" "+/-" "="

Возведение в степень
осуществляется с помощью кнопки "x n", сначала вводится число x, затем нажимается кнопка "x n", затем вводдится число степени n, равно. Например для вычисления "15 3" необходимо ввести 15, затем нажать кнопку "x n", затем ввести 3, нажать равно.

Вычисление квадратного корня
осуществляется введением числа и нажатием кнопки "√". Например для вычисления квадратного корня из 9 необходимо ввести 9 и нажать кнопку "√".
